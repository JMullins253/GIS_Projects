# GIS & Spatial Data Science Portfolio

Welcome to my repository of Geographic Information Systems (GIS) projects. This is an ongoing collection of work documenting my growth in spatial analysis, urban planning, and data visualization.

## Project Overview

This repository contains various geospatial analyses and mapping projects, focusing on:

* **Urban Analysis:** Evaluating city-level data to understand neighborhood density, resource distribution, and infrastructure.
* **Thematic Mapping:** Creating clear, data-driven visualizations using techniques like choropleth maps, treemaps, and heatmaps.
* **Spatial Statistics:** Utilizing quantitative methods to identify patterns, clusters, and correlations in geographic data.
* **Historical Geography:** Mapping the evolution of urban transit and infrastructure over time.

## Technology Stack

I leverage a variety of tools and libraries to perform analysis and create visualizations:

* **GIS Software:** ArcGIS Pro, QGIS, ArcGIS Online/StoryMaps
* **Programming:** Python (GeoPandas, Pandas, Matplotlib, ArcPy)
* **Data Analysis:** SQL, Excel, Spatial Statistics Tools
* **Visualization:** Tableau, Adobe Creative Suite, Leaflet/Mapbox
